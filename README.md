# Infuse Saver

**Infuse Save** — плагин для сохранения фильмов и сериалов из **Lampa** напрямую в **Infuse**.

Сслылка на плагин для Lampa: 
## Возможности
- Сохранение фильмов и сериалов из Lampa в Infuse
- Поддержка TorrServer / прямых ссылок
- Корректная работа с сезонами и сериями
- Быстрое добавление в медиатеку Infuse

## Назначение
Плагин создан для удобного переноса контента из Lampa в Infuse без ручного копирования ссылок.
